#Render checklist
  • >Runtime staat op Node.
  • >Start command is npm start.
  • >STAFF_PUBLIC_URL gebruikt de juiste domeinnaam.
  • >Discord redirect URI staat exact juist.

Zet secrets alleen in Render Environment Variables. Commit nooit .env.local, bot tokens of Supabase secret keys naar GitHub.
